Rule set for testers
Goal (primary win): Reveal all words (all letter-tiles that make up each detected word run).
Score: Lowest Clicks wins. Reveal = +1. Flag = +0. Guess letter: correct = +0, wrong = +3.
Bombs: If you Reveal a bomb 💣 you lose immediately (prototype behaviour). Flags are just markers.
Buoy/0: Empty tiles show adjacent bomb count (0–8). A 0 (🛟) floods and reveals connected EMPTY tiles only.
Letters: When revealed, each letter shows a 3×3 “compass”: remaining contiguous letters Up/Down/Left/Right.
New rule: When a word is completed, the tile(s) immediately before/after the word (end-caps) are auto-revealed as status (Option B):
• If an end-cap is a bomb, it shows 💣 but does not kill you (it’s reward intel).
• If an end-cap is empty, it reveals its number/buoy (and can flood EMPTY-only if it’s 0).
Hybrid bonuses (not required to win):
• Clean bombs = all bombs flagged correctly (⚑ on each bomb, no wrong flags).
• Full clear = all non-bomb tiles revealed (letters + empties).
